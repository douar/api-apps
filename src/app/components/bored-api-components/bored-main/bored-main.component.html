Bored? Let's find something to do!

The Bored API is a database of activities, and we tap into that to give you a random activity suggestion.
You can get random activity suggestions by pure randomness, by activity type (recreational, educational, etc), participant count, price, or price-range.

NOTICE:

Some activity suggestions will have a link to provide more information, some may not have one.
